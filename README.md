-> Ferramenta para calcular o IMC

Uma ferramenta para calcular o Índice de Massa Corporal (IMC) foi criada utilizando Python.

-> Informações sobre o projeto

Este projeto determina o IMC do usuário com base nas informações de peso e altura fornecidas,
e classifica o resultado em categorias de saúde.

-> Mecanismo de funcionamento

1. O usuário fornece seu peso em quilogramas.
2. O usuário fornece sua altura em metros.
3. O programa determina o IMC e apresenta a classificação.

-> Avaliações

| IMC | Categoria |
|---|---|
| Acima de 35 | Obesidade Mórbida |
| De 30 a 34 | Obesidade |
| De 25 a 29 | Sobrepeso |
| Faixa etária de 18 a 24 anos | Peso considerado saudável |
| Abaixo de 18 | Extrema magreza |
